# ParadoX Cinemas – Premium Movie Theatre Booking Website

**ParadoX Cinemas** is a responsive front-end web project designed to deliver a modern and premium movie theatre booking experience. Built using **HTML5, CSS3, Bootstrap 5, and JavaScript (Bootstrap Components)**, the project simulates a real-world cinema platform where users can explore movies, book tickets, access premium memberships, and share feedback through an elegant and user-friendly interface.

The website follows a premium black-and-gold cinematic theme to create a luxury theatre feel, focusing on both visual appeal and usability. It includes multiple interconnected pages such as Home, About, Movies, Ticket Booking, Premium Membership, and Customer Experience sections.

## Key Features

* Responsive Bootstrap-based design for desktop and mobile devices
* Dynamic homepage with hero carousel and “Now Showing” movie section
* Detailed movie listings with ratings, duration, and language selection
* Online ticket booking form with seat type and movie selection
* Premium membership subscription system with multiple plans
* Customer feedback section using collapse components
* Special offers section using Bootstrap offcanvas
* Premium theatre experience showcase with exclusive privileges
* About page highlighting theatre specialties and brand vision
* Modern dark theme UI with gold-accent premium styling

## Pages Included

* **Home Page** – Landing page with movie highlights and theatre specialties
* **About Page** – Theatre introduction, owner details, and gallery
* **Movies Page** – Movie listings with ratings and special features
* **Booking Page** – Ticket reservation form
* **Experience Page** – Feedback system, offers, and premium features
* **Membership Page** – Premium subscription registration form

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Components (Carousel, Collapse, Offcanvas, Forms, Cards, Alerts, Navbar, Tables)

## Project Objective

The main objective of this project is to create a visually attractive and functionally structured cinema website that demonstrates practical implementation of front-end web development concepts using Bootstrap. It showcases how real-world entertainment booking platforms can be designed with clean UI/UX principles and responsive layouts.
